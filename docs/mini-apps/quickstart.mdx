Get up and running with your first Mini App in just a few minutes. This guide will walk you through creating a simple "Hello World" Mini App that demonstrates the basic concepts and integration patterns.

## Prerequisites

Before you begin, make sure you have:

- Basic knowledge of HTML, CSS, and JavaScript
- A code editor (VS Code, Sublime Text, etc.)
- Node.js installed (version 16 or higher)
- A Base App developer account

## Step 1: Create Your Project

Start by creating a new directory for your Mini App:

```bash
mkdir my-mini-app
cd my-mini-app
npm init -y
```

### Create app.js

```javascript
class MiniApp {
    constructor() {
        this.isConnected = false;
        this.init();
    }

    init() {
        this.connectBtn = document.getElementById('connectBtn');
        this.statusDiv = document.getElementById('status');
        
        this.connectBtn.addEventListener('click', () => this.handleConnect());
        this.updateStatus('Ready to connect');
    }

    async handleConnect() {
        try {
            this.updateStatus('Connecting...');
            
            // Simulate wallet connection
            await this.simulateConnection();
            
            this.isConnected = true;
            this.connectBtn.textContent = 'Connected!';
            this.connectBtn.disabled = true;
            this.updateStatus('Successfully connected to wallet! ðŸŽ‰');
            
        } catch (error) {
            this.updateStatus(`Connection failed: ${error.message}`);
        }
    }

    async simulateConnection() {
        // Simulate network delay
        return new Promise((resolve) => {
            setTimeout(resolve, 1500);
        });
    }

    updateStatus(message) {
        this.statusDiv.textContent = message;
    }
}

// Initialize the app when the page loads
document.addEventListener('DOMContentLoaded', () => {
    new MiniApp();
});
```

## Step 3: Test Locally

Start a local development server:

```bash
# Using Python (if installed)
python -m http.server 8000

# Or using Node.js
npx http-server -p 8000
```

Open your browser and navigate to `http://localhost:8000` to see your Mini App in action.

## Step 4: Integrate with Base App

To integrate your Mini App with Base App, you'll need to:

### Add Base App Integration

Update your `app.js` to include Base App integration:

```javascript
class MiniApp {
    constructor() {
        this.isConnected = false;
        this.init();
        this.setupBaseAppIntegration();
    }

    setupBaseAppIntegration() {
        // Check if running in Base App
        if (window.baseApp) {
            this.baseApp = window.baseApp;
            this.updateStatus('Running in Base App! ðŸš€');
        } else {
            this.updateStatus('Running in standalone mode');
        }
    }

    async handleConnect() {
        try {
            this.updateStatus('Connecting...');
            
            if (this.baseApp) {
                // Use Base App wallet connection
                await this.connectToBaseApp();
            } else {
                // Fallback to simulation
                await this.simulateConnection();
            }
            
            this.isConnected = true;
            this.connectBtn.textContent = 'Connected!';
            this.connectBtn.disabled = true;
            this.updateStatus('Successfully connected! ðŸŽ‰');
            
        } catch (error) {
            this.updateStatus(`Connection failed: ${error.message}`);
        }
    }

    async connectToBaseApp() {
        // Base App specific connection logic
        if (this.baseApp.wallet) {
            await this.baseApp.wallet.connect();
        }
    }

    // ... rest of the class methods
}
```

## Step 5: Deploy Your App

### Option 1: Deploy to IPFS

```bash
# Install IPFS CLI tools
npm install -g ipfs

# Add your files to IPFS
ipfs add -r .

# Pin your files
ipfs pin add <your-hash>
```

### Option 2: Deploy to Traditional Hosting

- **Vercel**: Connect your GitHub repo and deploy automatically
- **Netlify**: Drag and drop your build folder
- **GitHub Pages**: Push to a GitHub repo and enable Pages
